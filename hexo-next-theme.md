# 下载主题

```
$ cd your-hexo-site
$ git clone https://github.com/iissnan/hexo-theme-next themes/next
```

#启用主题 

> 打开 站点配置文件，_config.yml 找到 theme 字段，并将其值更改为 next。

theme: next

# 主题设定

```
#scheme: Muse
#scheme: Mist
scheme: Pisces
```
